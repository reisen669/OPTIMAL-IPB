<objective>
Implement PalmEnsembleAlgorithm and register it in the OPTIMAL-IPB Processing provider.

The algorithm accepts any two palm centroid point layers (detector-agnostic) and merges them into a confidence-scored union: confidence=1.0 where both detectors agree within a distance threshold, confidence=0.5 where only one detects a palm. The source field records "both" or the user-supplied layer name for unmatched points.
