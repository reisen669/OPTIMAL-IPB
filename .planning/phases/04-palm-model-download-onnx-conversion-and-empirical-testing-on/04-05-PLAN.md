```python
from qgis.core import QgsExpressionContextUtils
key = QgsExpressionContextUtils.globalScope().variable('roboflow_api_key')
if key:
    print(f"API key found: {key[:4]}{'*' * (len(key)-4)}")  # print only first 4 chars
else:
    print("WARNING: roboflow_api_key global variable is NOT set")
```

If the key is NOT set:
- Go to Settings → Options → Variables (QGIS-level)
- Add a new variable: Name = `roboflow_api_key`, Value = (your Roboflow API key)
- Click OK, then re-run the snippet above to confirm it returns a non-empty value

**Step 2 — Register RoboflowPalmDetectionAlgorithm for this session:**
```python
import sys, os
PLUGIN_DIR = r'C:\Users\suily\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\optimal-ipb'
if PLUGIN_DIR not in sys.path:
    sys.path.insert(0, PLUGIN_DIR)

from roboflow_algorithm import RoboflowPalmDetectionAlgorithm
from qgis.core import QgsApplication
alg = RoboflowPalmDetectionAlgorithm()
reg = QgsApplication.processingRegistry()
if not reg.algorithmById('roboflow-palm-detection'):
    reg.addAlgorithm(alg)
    print("Roboflow algorithm registered as 'roboflow-palm-detection'")
else:
    print("Roboflow algorithm already registered")
```

**Roboflow on Rupat (~77 API calls, minimum 8s + network latency — run first):**
```python
import processing
PLUGIN_DIR = r'C:\Users\suily\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\optimal-ipb'
RUPAT = PLUGIN_DIR + r'\tif_online_samples\oam_rupat_indonesia_0.088mpx.tif'

result = processing.runAndLoadResults(
    'roboflow-palm-detection',
    {
        'INPUT': RUPAT,
        'CONFIDENCE': 0.25,
        'TYPE': 0,
        'OUTPUT': 'memory:'
    }
)
from qgis.core import QgsProject
layers = QgsProject.instance().mapLayersByName('Roboflow detections')
if not layers:
    layers = QgsProject.instance().mapLayersByName('Palm Trees')  # fallback
layer = layers[-1] if layers else None
if layer:
    count = layer.featureCount()
    print(f"Roboflow Rupat: n={count}")
else:
    print("Roboflow Rupat: output layer not found — check Log Messages panel for errors")
```

While Rupat runs, watch the QGIS Log Messages panel (View > Panels > Log Messages → select the QGIS log or Python Warning tab). The algorithm logs:
- "Processing tile N of M" (every tile)  — lets you estimate API call count
- Error messages if the API returns HTTP 4xx/5xx

After Rupat completes, note:
- detection count from console output
